# フロントエンド設計

## 技術スタック

| 技術         | バージョン              | 用途              |
| ------------ | ----------------------- | ----------------- |
| Vite         | 7.x                     | ビルドツール      |
| React        | 19.x                    | UI ライブラリ     |
| TypeScript   | 5.9                     | 型安全            |
| Tailwind CSS | 4.x                     | スタイリング      |
| shadcn/ui    | radix-vega スタイル     | UI コンポーネント |
| Tabler Icons | `@tabler/icons-react`   | アイコン          |
| M PLUS 1p    | Google Fonts CDN        | フォント (日本語) |
| React Router | 7.x                     | ルーティング      |
| Supabase JS  | `@supabase/supabase-js` | DB・Auth          |

### テンプレート構成

shadcn/ui 初期化済み:

- スタイル: `radix-vega`
- ベースカラー: `zinc` (oklch ベース)
- CSS 変数: 有効 (ライト/ダーク自動切替)
- アイコン: `tabler`
- `components.json` で `@/components`, `@/lib`, `@/hooks` のエイリアス設定済み

レガシー版の NavigationRail/BottomNav パターンを、shadcn/ui の `Sidebar` + レスポンシブ設計で再現。

```
┌──────────────────────────────────────────────────────────┐
│ ┌────────┐                                               │
│ │Sidebar │  メインコンテンツ                                 │

│ │ TiME   │  ┌─────────────────────────────────────┐      │
│ │        │  │                                     │      │
```

## 科目一覧画面 (`/`)

レガシー `ListScreen` に対応。

### ヘッダー

```
ポートレート:
┌──────────────────────────────────────┐
│  [専攻 Select]   [フィルター Button]    │
│  [検索バー (Command Palette)]          │
└──────────────────────────────────────┘

ランドスケープ:
┌──────────────────────────────────────┐
│  [専攻 Select]   [検索バー]             │
└──────────────────────────────────────┘
```

### 専攻セレクト (レガシー: ChoiceBox)

shadcn/ui `Select` コンポーネント。

レガシーではカリキュラムコード (16 項目) だったが、大学院では専攻 (12 項目) に変更:

| ラベル         | 値   | ラベル              | 値   |
| -------------- | ---- | ------------------- | ---- |
| 共通           | `00` | 電気・化学          | `05` |
| 機械(1)        | `01` | 共同原子力          | `06` |
| 機械(2)        | `02` | 建築都市デザイン(1) | `07` |
| 機械システム   | `03` | 建築都市デザイン(2) | `08` |
| エネルギー化学 | `04` | 情報                | `09` |
|                |      | 自然                | `11` |

> 正確な専攻一覧はカリキュラムコードスクレイピング時に確定

### フィルターパネル (レガシー: FilterColumn)

ポートレート: shadcn/ui `Sheet` (ボトムシート)
ランドスケープ: サイドバー (200px 固定幅)

| グループ | 選択肢                                                   | shadcn/ui           |
| -------- | -------------------------------------------------------- | ------------------- |
| 登録済み | チェックボックス 1 個                                    | `Checkbox`          |
| 空きコマ | チェックボックス 1 個                                    | `Checkbox`          |
| 学期     | 前期前/前期後/前期/前集中/後期前/後期後/後期/後集中/通年 | `Checkbox` グループ |
| 分類     | (シラバスから取得、専攻依存)                             | `Checkbox` グループ |
| 必選     | 必修/選択必修/選択                                       | `Checkbox` グループ |

> レガシーの「学年」フィルタは大学院ではほぼ 1 年のみのため削除。

#### フィルタロジック

- 各グループは AND 条件
- グループ内は OR 条件
- **何も選択されていないグループは無効化**（全件表示）

### 検索バー (レガシー: SearchBox)

shadcn/ui `Command` (cmdk ベース) を使用:

- ヒントテキスト: 「科目名・担当者で検索」
- オートコンプリート: 入力文字で部分一致
- 最大幅: 300px

### 科目カード (レガシー: CourseCard)

shadcn/ui `Card` ベース。

#### ポートレート

```
┌───────────────────────────────────────────┬──────┐
│ 前期後 月1・木1                               │      │
│ ロボティクス特論 (primary色、リンク)             │  [+] │
│                          対象[02機械] 選択 2.0単位 │      │
└───────────────────────────────────────────┴──────┘
```

#### ランドスケープ

```
┌──────┬─────────────────────┬─────────┬──────┬──────┐
│ 前期後 │ ロボティクス特論       │ 専門     │ 2.0  │ [登録] │
│ 月1   │ 佐藤 大祐            │ 選択    │ 単位  │       │
│ 木1   │                     │         │      │       │
└──────┴─────────────────────┴─────────┴──────┴──────┘
```

| 要素                | shadcn/ui                             |
| ------------------- | ------------------------------------- |
| カード              | `Card`                                |
| 登録ボタン (未登録) | `Button` variant=default, `IconPlus`  |
| 登録ボタン (登録済) | `Button` variant=outline, `IconCheck` |
| 科目名リンク        | シラバス URL への `<a>`               |

### 科目詳細ダイアログ (レガシー: CourseDialog)

shadcn/ui `Dialog`:

| 行  | ラベル     | 値                        |
| --- | ---------- | ------------------------- |
| 1   | 学期       | `前期後`                  |
| 2   | 曜日・時限 | `月1, 木1` (対開講)       |
| 3   | 担当者     | `佐藤 大祐`               |
| 4   | 分類       | `専門` (カリキュラム依存) |
| 5   | 必選       | `選択`                    |
| 6   | 単位数     | `2.0`                     |
| 7   | 講義コード | `smab020161`              |
| 8   | 教室       | `22A`                     |
| 9   | 受講対象   | `対象[02機械]`            |
| 10  | 備考       | (あれば表示)              |

アクション: `Button` 「登録」/ 「取消」

## 時間割画面 (`/timetable`)

レガシー `TableScreen` に対応。

### タブ切替

shadcn/ui `Tabs`:

| タブ | 表示要素                                                                     |
| ---- | ---------------------------------------------------------------------------- |
| 前期 | CourseTable(前半) + CourseTable(後半) + CourseWrap(通年/集中) + CreditsTable |
| 後期 | 同上 (後期データ)                                                            |

### 時限一覧 (レガシー: TimeTable)

```
| 1時限 9:20-11:00 | 2時限 11:10-12:50 | 3時限 13:40-15:20 | 4時限 15:30-17:10 | 5時限 17:20-19:00 |
```

shadcn/ui `Table` で横並び表示。

### 週間時間割グリッド (レガシー: CourseTable)

```
┌──────┬────┬────┬────┬────┬────┬────┐
│ 前半  │ 月  │ 火  │ 水  │ 木  │ 金  │ 土  │
├──────┼────┼────┼────┼────┼────┼────┤
│ 1時限 │    │    │    │    │    │    │
│ 2時限 │    │    │    │    │    │    │
│ 3時限 │    │    │    │    │    │    │
│ 4時限 │    │    │    │    │    │    │
│ 5時限 │    │    │    │    │    │    │
└──────┴────┴────┴────┴────┴────┴────┘
```

#### セルの状態 (レガシー: TableCard)

| 状態            | 表示              | 色                 |
| --------------- | ----------------- | ------------------ |
| 空              | 空白セル          | -                  |
| 科目 1 件       | 科目名 (2 行まで) | `secondary` 背景   |
| 重複 (2 件以上) | 「重複」          | `destructive` 背景 |

対開講: 同一科目は同じ色でハイライト。タップで `Dialog` 表示。

#### レスポンシブ

- ポートレート: 前半・後半を縦に配置
- ランドスケープ: 前半・後半を横に配置 (`grid-cols-2`)

### 通年・集中科目 (レガシー: CourseWrap)

`Card` 内にリスト表示。最小高さ 248px。

### 単位集計テーブル (レガシー: CreditsTable)

shadcn/ui `Table`:

#### 前期タブ

```
┌──────────┬──────────┬──────┬──────┐
│  カテゴリ  │  修得済   │ 前期  │ 合計  │
├──────────┼──────────┼──────┼──────┤
│ 専門      │ [Input]  │ 4.0  │ 14.0 │
│ 共通      │ [Input]  │ 2.0  │  6.0 │
│ 英語      │ [Input]  │ 0.0  │  2.0 │
├──────────┼──────────┼──────┼──────┤
│ 合計      │ 10.0     │ 6.0  │ 16.0 │
└──────────┴──────────┴──────┴──────┘
```

- 修得済み列: shadcn/ui `Input` (type=number, step=0.5)
- 前期タブのみ編集可、後期は「修得済+前期」で固定表示

## Sidebar / ナビゲーション

### Sidebar コンポーネント (デスクトップ)

shadcn/ui `Sidebar` を使用:

| セクション   | 要素                                                                                     |
| ------------ | ---------------------------------------------------------------------------------------- |
| **ヘッダー** | ロゴ: 折り畳み時「TiME」/ 展開時「TCU-TIME」                                             |
| **メニュー** | 科目一覧 (`IconList`)、時間割 (`IconTableFilled`)                                        |
| **フッター** | テーマ切替 (`IconSun`/`IconMoon`)、ユーザー (アバター/ログイン)、情報 (`IconInfoCircle`) |

### BottomNav (モバイル)

Tailwind で自作:

| 位置 | アイコン          | ラベル   |
| ---- | ----------------- | -------- |
| 左   | `IconList`        | 科目一覧 |
| 右   | `IconTableFilled` | 時間割   |

## 認証 (レガシー: Firebase Auth → Supabase Auth)

### ログイン画面

| プロバイダ     | shadcn/ui                 |
| -------------- | ------------------------- |
| メールアドレス | `Input` + `Button`        |
| Google         | `Button` with Google icon |

Supabase Auth UI (`@supabase/auth-ui-react`) を使用してもよい。

### プロフィール画面

| 要素           | 説明                       |
| -------------- | -------------------------- |
| ユーザー情報   | メール、表示名             |
| ログアウト     | `Button` variant=outline   |
| アカウント削除 | `AlertDialog` で確認後削除 |
